## Publish with GitHub Pages

This repo includes a GitHub Actions workflow for GitHub Pages. To publish your own public copy:

1. Create a new public GitHub repository.
2. Push this project to that repository.
3. In GitHub, open `Settings -> Pages` and set `Build and deployment` to `GitHub Actions`.
4. Push to `master` or run the `Deploy to GitHub Pages` workflow manually.

For a normal project repository, the site URL will be:

```txt
https://YOUR_GITHUB_NAME.github.io/YOUR_REPOSITORY_NAME/
```
